NYU Motorsports Telemetry


Telemetry platform for monitoring NYU’s Motorsports SAE BAJA competition vehicle. This platform supports reception and visualization of data from the competition vehicle along with several auxilary features. Each section of the platform and its functionality is explained below.

Dashboard

Lauches a data visualization tool in a new window. This tool queries the team's database and uses the return values to populate several graphs. These graphs can be configured to fit the team's needs. Graphs can also be added or deleted as new data sources become available.

Stopwatch

Time the race with the Stop Watch feature. This allows for overall time, as well as individual lap times, to be tracked in the user's browser.

Raw Telemety

This portal allows the raw data to be viewed. This is useful for debugging and understanding the models behind the data visualization portal. Data generated in the simulator will appear here, as well as any data from vehicle sensors.

Simulator

Generates mock vehicle data. Allows the user to experiment with different data patters on the visualization portal and raw telemetry portal.

CAN Decoder

Input CAN messages and view the return values. This portal sends user input to the CAN decoder API endpoint and returns the decoded JSON values. Useful for learning CAN's format and the differences between structured and CAN data.




New York University

New York University (NYU) is a private research university based in New York City. Founded in 1831, NYU's historical campus is in Greenwich Village, Lower Manhattan. NYU also has degree-granting campuses in Abu Dhabi and Shanghai, and academic centers in Accra, Berlin, Buenos Aires, Florence, London, Los Angeles, Madrid, Paris, Prague, Sydney, Tel Aviv, and Washington, D.C.
